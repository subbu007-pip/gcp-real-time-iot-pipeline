# GCP Real-Time IoT Pipeline

A real-time IoT streaming pipeline built using Google Cloud Platform services.

## Technologies Used
- Google Cloud Pub/Sub
- Cloud Functions
- BigQuery
- Cloud Monitoring
- Cloud Build CI/CD
- Python
- ESP32 / Arduino

## Features
- Real-time IoT telemetry ingestion
- Event-driven serverless processing
- BigQuery analytics storage
- Monitoring and dashboards
- Automated CI/CD deployment

## Architecture
ESP32 Sensors → Pub/Sub → Cloud Functions → BigQuery → Looker Studio
